# dns-proxy

DNS拦截与代理转发服务

核心功能

1. DNS拦截服务
• 监听DNS查询，根据GFWList规则匹配域名
• 匹配域名返回自定义IP实现流量劫持
• 未匹配域名转发至上游DNS正常解析

2. 透明代理服务
• 监听HTTP80和HTTPS443端口
• 解析Host头和TLS的SNI获取目标域名
• 支持SOCKS5/HTTP代理转发

3. Web管理界面
• 规则管理：搜索、增删改域名规则
• 配置管理：可视化配置DNS、代理参数
• 实时监控：WebSocket推送查询和转发日志

应用场景

将设备DNS指向本服务后，指定域名流量被劫持到本机代理端口，再通过SOCKS5/HTTP代理转发，实现特定域名走代理、其他直连的分流效果，常用于科学上网或企业网络管理.

技术特点
• 支持代理认证、配置持久化
• 服务状态自动恢复
• HTTP/2优化的TCP转发
